Heads up

    
There is no interrupt in this interface — a runaway proof
    (e.g. a non-terminating REWRITE_CONV) cannot be
    cancelled.  Ctrl+K wipes the toplevel and re-boots
    the kernel from scratch (~1–2 min), so use it only as a last
    resort.

    
The search command is not available here:
    update_database.ml is intentionally not shipped
    with this build (so loadt "update_database.ml";;
    will fail with a file-not-found).

About

    
This is HOL Light running entirely in your
      browser.  The OCaml toplevel and HOL Light kernel are compiled
      to JavaScript with
      js_of_ocaml
      and run in a Web Worker; the page just streams its output and
      forwards your input.  No proofs are sent to a server.

    
Loading extra libraries.
      loadt "Library/words.ml";; (and loads,
      needs) fetch the file over HTTP from this page's
      mirror of the
      jrh13/hol-light
      repo, then evaluate it in the toplevel with output streaming live.
      The first fetch is cached, so re-loading the same file is a no-op.
